Eururalis 2.0 interactive tool Note

The interactive tool offers more functionality than just the contents: maps and graphs can be explored for different scenarios and for different policy options.

  1. Find out whether you have Java 1.6 installed (JRE) or even a higher version. To do so, get a command prompt and enter:
    java -version
  2. If you don't have Java 1.6 installed, look for the Java Runtime Environment (JRE) 6.0 Update 3 or higher here
In April 2011 a warning started appearing at startup, as a result of our decision not to renew the certificate again. It became necessary to abandon the JNLP protocol. We made it possible for you to still install the application though - but now directly onto your computer. You do so at your own risk. Download the installer here (for Windows).
 
Once installed, you can start the application by clicking doubleclicking the Eururalis2 shortcut on your desktop.

Please be aware that you need a fast Internet connection because of more than 560 MB of data. In addition you will need disk space:
  • approx. 740 MB of disk space to store those data in a folder of your choice; and
  • approx. 25 MB for storing the program files
